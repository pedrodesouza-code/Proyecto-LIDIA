# Archivo historico de documentacion

Esta carpeta conserva documentos que ya no son fuente de verdad operativa.

## Uso correcto

- Sirven como evidencia historica, respaldo de proceso o material de trabajo.
- No deben usarse para afirmar el estado actual del sistema.
- Para defensa EC3 usar los documentos vigentes ubicados en la raiz de `docs/`.

## Subcarpetas

| Carpeta | Contenido |
|---|---|
| `historico/` | Documentos antiguos, entregas previas, guias completas anteriores y evidencia UTEC de cortes anteriores |
| `desarrollo/` | Bitacoras, guias internas, planes y material operativo de desarrollo |
| `docx/` | Versiones Word generadas para entregas o borradores |

Fuente actual recomendada:

```text
docs/README.md
docs/UNIFICACION_EC1_EC2_EC3_FOCO_EC3_2026-05-22.md
docs/GUIA_DEFENSA_TECNICA_COMPLETA_2026-05-22.md
docs/CIERRE_ENTREGA_2026-05-22.md
docs/ESTADO_ACTUAL_PROYECTO_2026-05-22.md
```
